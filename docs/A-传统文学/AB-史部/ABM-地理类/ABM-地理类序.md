# 地理类

## 序言

- **内容：**
  - 极为丰富，是所有关于“地”的记录。

- **下分小属：**
  - **宫殿疏：** 记载宫廷建筑，如《三辅黄图》。

  - **总志：** 全国性地理志，如《元和郡县志》、《大明一统志》。

  - **都会郡县：** 地方志，如省、府、州、县志。

  - **河渠：** 水道水利，如《水经注》。

  - **边防：** 边疆防务与地理。

  - **山川：** 名山记，如《庐山记》。

  - **古迹：** 文物古迹考，如《金石录》。

  - **杂记：** 游记、风土记，如《桂林风土记》。

  - **游记：** 如《徐霞客游记》。

  - **外纪：** 外国地理，如《大唐西域记》。
