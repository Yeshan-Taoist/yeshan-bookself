# 曲类

## 序言

- **内容：**
  - 收录曲的作品集及评论。

- **音乐基础：**
  - 北曲、南曲（新兴的民间俗乐）

- **语言风格：**
  - 通俗活泼，直白泼辣

- **句式结构：**
  - 有曲牌，但可加**衬字**，句式更灵活

- **内容题材：**
  - 题材广阔，叹世、归隐、讥时、情爱皆可

- **代表体裁：**
  - 分为**散曲**（诗歌性质）和**剧曲**（戏曲唱词）

- **代表作家：**
  - 关汉卿、马致远、张可久

- **代表作品：**
  - 《天净沙·秋思》（散曲）、《窦娥冤》（剧曲）
