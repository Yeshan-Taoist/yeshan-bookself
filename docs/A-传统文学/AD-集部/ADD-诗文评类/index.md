## 序言

- **内容：**
  - 文学理论和文学批评著作，包括对作家、作品、文学流派、创作方法的评论和研究。

- **特点：**
  - 这是中国古代的“文学概论”和“文艺学”。

- **代表典籍：**
  - **《文心雕龙》**（刘勰著，体大思精的文学理论巨著）

  - **《诗品》**（钟嵘著，品评诗歌的著作）

  - **《沧浪诗话》**（严羽著，以禅论诗的经典）

  - **《随园诗话》**（袁枚著，性灵派诗论代表）
