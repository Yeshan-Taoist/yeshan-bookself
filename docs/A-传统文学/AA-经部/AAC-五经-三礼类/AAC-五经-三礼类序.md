# 三礼类

## 序言

- 内容：
  - 研究与阐释“三礼”的著作。

- 核心典籍：
  - 《周礼》、《仪礼》、《礼记》。

- 说明：
  - 此类别非常繁复，下分 6 个小属：
    - **周礼之属：**关于《周礼》（讲官制和政治制度）的著作。

    - **仪礼之属：**关于《仪礼》（讲具体行为规范和典礼仪式）的著作。

    - **礼记之属：**关于《礼记》（阐释礼的意义和理论）的著作。

    - **三礼总义之属：**综合论述“三礼”的著作。

    - **通礼之属：**贯通历代礼制的著作，如《礼书通故》。

    - **杂礼书之属：**私家所撰关于民间礼仪的著作。
