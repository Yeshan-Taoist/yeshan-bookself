# 小学类

## 序言

- **内容：**
  - 语言文字学著作，是读经、通经的基础工具。

- **核心典籍：**
  - 《尔雅》、《说文解字》、《广韵》等。

- **下分 4 个小属：**
  - **训诂之属：** 解释古书中词句意义的著作，如《尔雅》。

  - **字书之属：** 讲解文字形体、音义及用法的著作，如《说文解字》。

  - **韵书之属：** 分韵编排，说明文字读音的著作，如《广韵》、《平水韵》。

  - **知识导读：** 如《三字经》、《百家姓》、《千字文》等。
